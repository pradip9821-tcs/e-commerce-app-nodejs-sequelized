# e-commerce-app-nodejs-sequelize

This is an e-commerce node app and in this app mysql/sequelize are used as a database.

SetUp instructions :-

-> First of all install Node and npm on your system (Visual Studio Code is recommended for coding).
-> In terminal or cmd,create your own node_package using command 'npm init'.
-> After creating package you have to install node libraries listed in package.txt.
